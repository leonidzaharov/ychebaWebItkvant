HTML — Ссылки и картинки
Фишки + практика, чтобы было интересно

Зачем этот урок?

В шпаргалке уже есть “что такое <a> и <img>”. Тут мы делаем приёмы: кнопка-ссылка, прыжок по странице, картинка-ссылка, подпись к картинке и маленький квест.

Важно: ссылки на свои файлы делай через ./ или ../, а не через /, чтобы не ломалось на GitHub Pages.
1) Ссылка-кнопка 2) Якорь (прыжок) 3) Картинка + подпись 4) Картинка-ссылка Мини-квест

1) Ссылка-кнопка (вау-эффект)

Это обычная ссылка <a>, но выглядит как кнопка. Нажал — открылся файл/сайт.

<a class="btnMini" href="./cheats/html-cheatsheet.html" target="_blank" rel="noopener">
  Открыть шпаргалку
</a>
Почему rel="noopener"? Это безопаснее, когда открываешь новую вкладку через target="_blank".
Открыть шпаргалку Открыть сайт
Задание: сделай свою кнопку “Мой проект” и веди её на любой файл ./lessons/...

2) Якорь: прыжок по странице

Якорь — это когда ссылка ведёт не на другой сайт, а на место в этой же странице.

<a href="#quest">Прыгнуть к мини-квесту ↓</a>

...

<section id="quest">
  ...
</section>
Прыгнуть к мини-квесту ↓
Задание: сделай ссылку “Наверх ↑” (под конец страницы) и веди её на #top (создай id="top" в начале).

3) Картинка + подпись (как у взрослых)

Тег <figure> — блок для картинки, а <figcaption> — подпись снизу.

<figure>
  <img src="../img/your-image.png" alt="Моя картинка" width="420">
  <figcaption>Мой первый проект</figcaption>
</figure>
alt нужен всегда: если картинка не загрузится — останется текст (и это полезно).

4) Картинка-ссылка (кликаешь по картинке)

Очень простая магия: оборачиваем <img> в <a>.

<a href="https://example.com" target="_blank" rel="noopener">
  <img src="../img/your-image.png" alt="Нажми на картинку" width="360">
</a>
Пример
Нажми по картинке → откроется сайт.
Фишка
Картинка может вести на твой файл, например: ./html-skeletons.html
Открыть скелеты Вернуться к кнопке ↑
Задание: сделай картинку-ссылку, которая ведёт на твою шпаргалку или на любой урок.
