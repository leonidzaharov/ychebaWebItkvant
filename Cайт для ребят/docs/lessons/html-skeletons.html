HTML — 4 скелета сайта
Скопируй → вставь → поменяй текст → готово

Как пользоваться

Выбери пример ниже, скопируй код, вставь в новый файл .html и меняй слова/картинки.

Подсказка: файл лучше называть латиницей без пробелов, например: site-01.html
1) Минимум 2) Визитка 3) Карточки 4) Большой

1) Минимальный скелет страницы

Самое главное: заголовок, текст, кнопка.

Внутри <body> — всё, что видно на странице. 
<!doctype html>
<html lang="ru">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>Мой первый сайт</title>
</head>

<body>
  <h1>Привет!</h1>
  <p>Это мой первый сайт.</p>

  <button>Это кнопка</button>
</body>
</html>
Задание: поменяй заголовок и текст. Сделай кнопку: “Нажми меня”.

2) Скелет “Визитка” (header + main + footer)

Уже похоже на нормальный сайт: сверху меню, в середине блоки, снизу подвал.

<!doctype html>
<html lang="ru">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>Сайт-визитка</title>
</head>

<body>
  <header>
    <h1>Мой клуб</h1>
    <nav>
      <a href="#about">О нас</a> |
      <a href="#projects">Проекты</a> |
      <a href="#contacts">Контакты</a>
    </nav>
  </header>

  <main>
    <section id="about">
      <h2>О нас</h2>
      <p>Мы учимся делать сайты и игры.</p>
    </section>

    <section id="projects">
      <h2>Проекты</h2>
      <ul>
        <li>Сайт-визитка</li>
        <li>Карточки с анимацией</li>
        <li>Мини-игра</li>
      </ul>
    </section>

    <section id="contacts">
      <h2>Контакты</h2>
      <p>Напиши нам: <a href="mailto:test@example.com">test@example.com</a></p>
    </section>
  </main>

  <footer>
    <p>© 2026 Мой клуб</p>
  </footer>
</body>
</html>
Задание: добавь ещё 1 пункт в список проектов и поменяй email.

3) Скелет “Карточки-разделы” (как у нас на главной)

Удобно для учебного сайта: 3–6 карточек, каждая ведёт на тему/урок.

Скопируй этот скелет 
<!doctype html>
<html lang="ru">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>Мои темы</title>
  <style>
    body{ font-family: Arial; margin:20px; }
    .grid{ display:grid; grid-template-columns: repeat(auto-fit, minmax(200px,1fr)); gap:12px; }
    .card{ border:1px solid #ccc; border-radius:12px; padding:12px; text-decoration:none; color:#111; }
    .card:hover{ transform: translateY(-2px); }
    .muted{ color:#555; font-size: 13px; }
  </style>
</head>

<body>
  <h1>Темы</h1>
  <p class="muted">Нажми на карточку, чтобы открыть тему.</p>

  <div class="grid">
    <a class="card" href="lesson-1.html" target="_blank">
      <b>HTML</b>
      <div class="muted">Теги, ссылки, картинки</div>
    </a>

    <a class="card" href="lesson-2.html" target="_blank">
      <b>CSS</b>
      <div class="muted">Цвета, отступы, анимации</div>
    </a>

    <a class="card" href="lesson-3.html" target="_blank">
      <b>JavaScript</b>
      <div class="muted">Условия, циклы, игры</div>
    </a>
  </div>
</body>
</html>
Задание: добавь 4-ю карточку “Шпаргалка” и сделай ссылку на свою шпаргалку.

4) Большой скелет “Мини-лендинг” (много блоков)

Это уже “похож на настоящий сайт”: герой-блок, карточки, таблица, форма.

Скопируй большой скелет 
<!doctype html>
<html lang="ru">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>Мини-лендинг</title>

  <style>
    body{ margin:0; font-family: Arial; background:#0b1022; color:#fff; }
    header{ padding:16px 20px; background: rgba(0,0,0,.35); position: sticky; top:0; }
    a{ color:#7cddff; }
    .wrap{ max-width: 1000px; margin: 0 auto; padding: 20px; }
    .hero{ border:1px solid rgba(255,255,255,.18); border-radius:16px; padding:20px; background: rgba(255,255,255,.06); }
    .btn{ display:inline-block; padding:10px 12px; border-radius:12px; background:#7c5cff; color:#fff; text-decoration:none; font-weight:700; }
    .grid{ display:grid; grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); gap:12px; margin-top: 14px; }
    .card{ border:1px solid rgba(255,255,255,.18); border-radius:16px; padding:14px; background: rgba(255,255,255,.06); }
    table{ width:100%; border-collapse: collapse; margin-top: 12px; }
    th,td{ border:1px solid rgba(255,255,255,.18); padding:10px; text-align:left; }
    input,button{ padding:10px; border-radius:12px; border:1px solid rgba(255,255,255,.25); background: rgba(0,0,0,.25); color:#fff; }
    .row{ display:flex; gap:10px; flex-wrap:wrap; }
  </style>
</head>

<body>
  <header>
    <b>Мой сайт</b> •
    <a href="#topics">Темы</a> •
    <a href="#table">Таблица</a> •
    <a href="#form">Форма</a>
  </header>

  <main class="wrap">
    <section class="hero">
      <h1>Учимся как в игре</h1>
      <p>Скопируй шаблон и поменяй текст, чтобы получилось “твоё”.</p>
      <a class="btn" href="#topics">Начать ↓</a>
    </section>

    <section id="topics">
      <h2>Темы</h2>
      <div class="grid">
        <div class="card"><b>HTML</b><div>теги, ссылки</div></div>
        <div class="card"><b>CSS</b><div>цвета, отступы</div></div>
        <div class="card"><b>JS</b><div>логика, игры</div></div>
      </div>
    </section>

    <section id="table">
      <h2>Таблица уроков</h2>
      <table>
        <tr><th>Тема</th><th>Сложность</th></tr>
        <tr><td>Теги</td><td>Легко</td></tr>
        <tr><td>Ссылки</td><td>Легко</td></tr>
        <tr><td>Таблицы</td><td>Средне</td></tr>
      </table>
    </section>

    <section id="form">
      <h2>Форма (пример)</h2>
      <p>Это как “анкета”. Пока без JavaScript.</p>

      <div class="row">
        <input type="text" placeholder="Имя" />
        <input type="number" placeholder="Возраст" />
        <button>Отправить</button>
      </div>
    </section>

    <footer style="margin-top:18px; opacity:.8;">
      © 2026 Мой сайт
    </footer>
  </main>
</body>
</html>
Задание: поменяй цвета, названия карточек и добавь ещё одну строку в таблицу.
