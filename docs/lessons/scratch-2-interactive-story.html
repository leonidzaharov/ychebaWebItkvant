Интерактивная история Начинающий

Создай мультик-историю, в которой зритель может влиять на сюжет! Персонажи разговаривают, перемещаются между локациями, а в ключевых моментах зритель выбирает, что будет дальше.

Чему научишься

  • Создавать сценарий с несколькими сценами
  • Использовать сообщения (передать/получить) для переключения сцен
  • Работать с блоком «спросить и ждать» для ввода от пользователя
  • Применять условия (если/иначе) для ветвления сюжета
  • Управлять видимостью спрайтов (показаться/спрятаться)
  • Синхронизировать действия нескольких персонажей

План реализации

  1. Придумай сюжет: Напиши короткую историю с 3-5 сценами. Придумай минимум 1 точку выбора, где зритель решает, что будет дальше (например: «Пойти налево или направо?»).
  2. Нарисуй раскадровку: На бумаге нарисуй каждую сцену — какой фон, какие персонажи, что они говорят.
  3. Создай фоны: Сделай отдельный фон для каждой локации (лес, замок, город, космос — что подходит к твоей истории).
  4. Добавь персонажей: Создай 2-3 персонажа. Для каждого настрой начальную позицию, размер, костюмы для разных эмоций.
  5. Запрограммируй диалоги: Используй блоки «говорить ... секунд» для реплик. Между репликами добавь паузы «ждать». Синхронизируй персонажей через «передать сообщение».
  6. Добавь выбор: В ключевом моменте используй «спросить ... и ждать». Через «если ответ = ... то» направь историю по разным веткам.
  7. Переходы между сценами: Используй «передать сообщение» для смены фона и появления/исчезания нужных персонажей.
  8. Финал и титры: Сделай красивую концовку. Добавь титры — кто создал историю.

Критерии оценки

  • Минимум 3 сцены (разные фоны/локации)
  • Минимум 2 персонажа с диалогами
  • Есть хотя бы 1 точка выбора (зритель влияет на сюжет)
  • Сцены переключаются через сообщения
  • История имеет начало, развитие и конец
  • Нет ошибок — всё работает от зелёного флажка до финала

Идеи для улучшения

  • Добавь 2-3 точки выбора с разными концовками
  • Озвучь персонажей своим голосом
  • Добавь фоновую музыку, разную для каждой сцены
  • Сделай анимацию переходов между сценами (затемнение/осветление)
