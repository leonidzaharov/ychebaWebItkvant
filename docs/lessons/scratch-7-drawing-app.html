Рисовалка Продвинутый

Создай полноценное приложение для рисования! Мышью рисуешь на холсте, выбираешь цвет из палитры, меняешь размер кисти, можешь очистить холст. Как настоящий Paint, только в Scratch!

Чему научишься

  • Работать с расширением «Перо» (опустить/поднять перо, стереть всё)
  • Отслеживать позицию мыши и нажатие кнопки мыши
  • Создавать пользовательский интерфейс из спрайтов (кнопки, палитра)
  • Управлять цветом и размером пера через переменные
  • Использовать сообщения для связи между кнопками и кистью
  • Программировать инструменты (кисть, ластик, заливка)

План реализации

  1. Спрайт-кисть: Создай маленький спрайт-точку (или курсор). В бесконечном цикле: «перейти к указателю мыши». Подключи расширение «Перо».
  2. Рисование мышью: «Если мышь нажата» → «опустить перо», иначе → «поднять перо». Теперь при зажатой мыши рисуется линия!
  3. Палитра цветов: Создай 6-8 спрайтов-кружков разных цветов (красный, синий, зелёный, жёлтый и т.д.). При клике на кружок — меняй цвет пера: «установить цвет пера».
  4. Размер кисти: Создай кнопки «+» и «-» (спрайты). При клике: «изменить размер пера на 2» или «на -2». Покажи текущий размер в переменной.
  5. Кнопка «Очистить»: Создай спрайт-кнопку «Очистить». При клике: «стереть всё» — холст очищается.
  6. Ластик: Создай кнопку «Ластик». При клике: установи цвет пера = цвет фона (белый). Кнопка «Кисть» возвращает выбранный цвет.
  7. Ограничение области: Рисовать можно только в области холста (не по кнопкам). Добавь проверку: если курсор в зоне палитры — поднять перо.
  8. Оформление интерфейса: Расположи палитру слева или сверху, кнопки — снизу. Сделай рамку холста. Добавь подписи к кнопкам.

Критерии оценки

  • Рисование мышью работает (зажал — рисуешь, отпустил — нет)
  • Минимум 6 цветов в палитре
  • Можно менять размер кисти (больше/меньше)
  • Есть кнопка «Очистить холст»
  • Есть ластик
  • Интерфейс понятный, кнопки не мешают рисованию

Идеи для улучшения

  • Добавь инструмент «штамп» — ставит готовые фигуры (звезда, сердце)
  • Сделай инструмент «линия» (клик → клик = прямая линия)
  • Добавь «отмену» последнего действия
  • Сделай режим «калейдоскоп» — рисуешь симметрично в 4 стороны
