Космический шутер Продвинутый

Создай космическую стрелялку! Твой корабль летит в космосе, стреляет лазерами по волнам пришельцев. Враги атакуют в ответ, есть здоровье, бонусы и эпичный финальный босс!

Чему научишься

  • Создавать снаряды через клоны (стрельба: пробел → клон пули летит вверх)
  • Программировать волны врагов (клоны врагов с разным поведением)
  • Определять столкновения между клонами (пуля попала во врага)
  • Работать с несколькими переменными (HP игрока, HP врага, очки, волна)
  • Создавать босс-файт (особый враг с большим HP и атаками)
  • Управлять сложной игровой логикой (состояния игры, паузы, переходы)

План реализации

  1. Корабль игрока: Нарисуй спрайт корабля. Управление стрелками (влево/вправо внизу экрана). Можно добавить движение вверх/вниз в небольшом диапазоне.
  2. Стрельба: Создай спрайт «Пуля». При нажатии пробела: создать клон пули в позиции корабля. «Когда я начинаю как клон»: двигаться вверх (изменять y на 10), если y > 170 — удалить клон.
  3. Враги: Создай спрайт «Враг» с несколькими костюмами (разные типы). Через сообщения запускай волны: создавай N клонов. Каждый клон появляется сверху в случайной позиции, медленно спускается вниз.
  4. Попадание пули во врага: В скрипте клона врага: «если касается [Пуля]» → уменьшить HP врага, если HP=0 → анимация взрыва, +очки, удалить клон. В скрипте пули: «если касается [Враг]» → удалить клон пули.
  5. Атака врагов: Враги тоже стреляют! Создай спрайт «Вражеская пуля». Клоны врагов периодически создают клон вражеской пули, которая летит вниз. При касании корабля — урон игроку.
  6. Здоровье игрока: Переменная «HP» (начало: 3). При попадании вражеской пули: HP-1, мигание корабля. При HP=0 — Game Over.
  7. Волны: Переменная «Волна». Волна 1: 5 простых врагов. Волна 2: 8 врагов, быстрее. Волна 3: босс! Между волнами — пауза с текстом «Волна 2!».
  8. Босс: Особый спрайт с большим HP (например, 20). Двигается влево-вправо сверху, стреляет часто. При уничтожении — победа!
  9. Бонусы: Изредка падают бонусы: +1 HP (лечение), двойная стрельба (на 10 секунд), щит (на 5 секунд).

Критерии оценки

  • Корабль управляется стрелками, стреляет по пробелу
  • Пули реализованы через клоны
  • Враги появляются волнами, двигаются и могут стрелять
  • Есть определение попаданий (пуля → враг, враг → игрок)
  • Есть HP, очки и номер волны
  • Есть минимум 3 волны и финальный босс
  • Есть экраны Game Over и Victory

Идеи для улучшения

  • Добавь прокручивающийся фон (звёзды летят вниз — эффект полёта)
  • Сделай магазин между волнами (за очки покупаешь улучшения)
  • Добавь разные типы оружия (лазер, ракеты, тройной выстрел)
  • Создай систему «комбо» — за серию попаданий подряд множитель очков растёт
