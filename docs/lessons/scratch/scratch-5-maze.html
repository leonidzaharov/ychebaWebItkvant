Лабиринт Средний

Создай игру, где персонаж должен пройти лабиринт, не касаясь стен! Управление стрелками, есть таймер, ловушки и несколько уровней. Дошёл до выхода — переход на следующий уровень!

Чему научишься

  • Управлять спрайтом с клавиатуры (стрелки вверх/вниз/влево/вправо)
  • Определять столкновения с цветом (касается цвета стены?)
  • Создавать несколько уровней (смена фонов-лабиринтов)
  • Работать с таймером и переменными
  • Программировать условие победы (достиг выхода)
  • Использовать начальную позицию и сброс при столкновении

План реализации

  1. Нарисуй лабиринт: Создай фон с лабиринтом. Стены — один цвет (например, чёрный), проходы — другой. Отметь старт и финиш. Сделай 3 разных фона для 3 уровней.
  2. Создай персонажа: Нарисуй маленького персонажа (шарик, мышку, робота). Важно: он должен быть маленьким, чтобы помещаться в проходах.
  3. Управление стрелками: «Когда клавиша [стрелка вверх] нажата» → «изменить y на 3». Аналогично для всех 4 направлений. Скорость подбери так, чтобы было удобно.
  4. Столкновение со стенами: В цикле «всегда» проверяй: «если касается цвета [чёрный]» → верни персонажа на старт (или сдвинь назад). Добавь звук столкновения.
  5. Условие победы: «Если касается цвета [зелёный]» (цвет выхода) → переход на следующий уровень. Смени фон, верни персонажа на старт нового уровня.
  6. Таймер: Создай переменную «Время». Каждую секунду увеличивай на 1. Покажи на экране. После финиша — покажи итоговое время.
  7. Уровни: Сделай 3 уровня с нарастающей сложностью. Первый — простой, второй — узкие проходы, третий — с тупиками.
  8. Экран победы: После прохождения всех уровней покажи экран «Победа!» с итоговым временем.

Критерии оценки

  • Минимум 3 уровня (3 разных лабиринта)
  • Управление стрелками работает плавно
  • Столкновение со стенами корректно обрабатывается
  • Есть таймер, показывающий время прохождения
  • При достижении выхода — переход на следующий уровень
  • Есть экран победы в конце

Идеи для улучшения

  • Добавь движущиеся ловушки (спрайты-враги, которые патрулируют)
  • Добавь бонусы-монетки внутри лабиринта (собираешь — бонус к очкам)
  • Сделай «туман войны» — видно только область вокруг персонажа
  • Добавь второго игрока (WASD) для режима «кто быстрее»
