RPG-приключение Экспертный

Создай настоящую ролевую игру с открытым миром! Герой исследует карту, общается с NPC, выполняет квесты, сражается с монстрами, собирает предметы в инвентарь и прокачивается. Самый масштабный проект!

Чему научишься

  • Создавать систему локаций (карта мира из нескольких экранов)
  • Программировать NPC с диалогами (ветвящиеся разговоры, квесты)
  • Работать с инвентарём (список предметов, использование, экипировка)
  • Создавать боевую систему (пошаговую или в реальном времени)
  • Управлять переменными персонажа (HP, атака, защита, уровень, опыт)
  • Использовать списки для хранения данных (инвентарь, квесты, диалоги)
  • Программировать систему квестов (получить → выполнить → сдать → награда)
  • Создавать сложную игровую архитектуру из множества взаимосвязанных систем

План реализации

  1. Дизайн мира: На бумаге нарисуй карту мира: 4-6 локаций (деревня, лес, пещера, замок и т.д.). Для каждой локации — отдельный фон. Продумай, как локации связаны (выход слева → лес, выход справа → город).
  2. Персонаж и навигация: Создай персонажа с костюмами для 4 направлений (вверх, вниз, влево, вправо) + анимация ходьбы. Управление стрелками. При достижении края экрана — смена фона (переход в другую локацию).
  3. Переменные героя: Создай: HP (здоровье), maxHP, Атака, Защита, Уровень, Опыт, Золото. Отображай HP и Золото на экране. При получении опыта — проверяй повышение уровня.
  4. NPC и диалоги: Создай 3-4 NPC (спрайты). При подходе и нажатии E — диалог. Используй списки для хранения реплик. Некоторые NPC дают квесты: «Принеси мне 3 гриба из леса».
  5. Система квестов: Список «Активные квесты». При получении квеста — добавь в список. При выполнении условия — отметь как выполненный. При разговоре с NPC-квестодателем — проверяй, выполнен ли квест, и давай награду.
  6. Инвентарь: Список «Инвентарь». При подборе предмета — добавить в список. Создай экран инвентаря (отдельный фон или оверлей): при нажатии I — показать/скрыть инвентарь с кнопками «Использовать».
  7. Боевая система: При столкновении с монстром — переход в режим боя. Пошаговый: «Атаковать / Защищаться / Использовать предмет». Урон = Атака героя - Защита монстра. Монстр атакует в ответ. При победе — опыт + золото + дроп предметов.
  8. Монстры: Создай 3-4 типа монстров (слабые в лесу, средние в пещере, сильные в замке). У каждого свои HP, атака, защита. Используй списки для хранения характеристик.
  9. Прокачка: При наборе нужного опыта — повышение уровня: HP +5, Атака +2, Защита +1. Показывай сообщение «Уровень повышен!».
  10. Финальный босс: В последней локации — босс. У него много HP и сильные атаки. Для победы нужно прокачаться и собрать хорошие предметы. После победы — финальная заставка и титры.

Критерии оценки

  • Минимум 4 локации с переходами между ними
  • Персонаж с анимацией ходьбы в 4 направлениях
  • Минимум 2 NPC с диалогами
  • Есть хотя бы 1 квест (получить → выполнить → сдать)
  • Работающая боевая система
  • Есть инвентарь с предметами
  • Есть система HP, атаки и уровней
  • Есть финальный босс и концовка

Идеи для улучшения

  • Добавь магазин, где можно купить оружие и зелья за золото
  • Сделай систему экипировки (меч, щит, броня — влияют на характеристики)
  • Добавь мини-карту в углу экрана
  • Создай несколько концовок в зависимости от выполненных квестов
  • Добавь музыку для каждой локации и боёв
