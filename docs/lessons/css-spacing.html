CSS — Отступы
margin / padding — мини-игра “попади блоком в рамку”

1) Главное правило

margin — снаружи padding — внутри

margin — отступ между блоками
padding — “воздух” внутри блока
.card{
  margin: 12px;   /* снаружи */
  padding: 16px;  /* внутри */
}
Если блок “слишком прижат” к краям — добавь padding. Если блок “слишком близко” к другим — добавь margin.

2) Мини-игра: “попади блоком в рамку”

Двигай margin (снаружи) и меняй padding (внутри). Твоя цель — сделать так, чтобы блок выглядел “красиво”: он не прилип к краям, и текст внутри не прижат.

Рамка (target)
Я — блок Меня двигает margin
А это внутренний блок (видно padding)
✅ Отлично! Похоже на аккуратный блок: есть воздух снаружи и внутри.
Подсказка: попробуй сделать margin 18–26, а padding 14–22. Потом поэкспериментируй!

Настройки

0px
0px
12px
Как запомнить: padding — “подушка” внутри, margin — “расстояние до других”.

3) Шпаргалка сокращений

В CSS можно писать коротко.

/* 1 значение */
margin: 20px;        /* везде 20 */

/* 2 значения: (верх/низ) (лево/право) */
margin: 10px 30px;   /* вверх-вниз 10, лево-право 30 */

/* 4 значения: верх право низ лево */
padding: 8px 12px 16px 12px;
Быстрая проверка: если 2 значения — это “вверх/низ” и “лево/право”.

4) Задания (на 3–7 минут)

Задание 1: сделай блок так, чтобы он не касался рамки и выглядел аккуратно.
Задание 2: поставь margin-left больше, чем margin-top (блок уедет вправо).
Задание 3: поставь padding = 0 и посмотри, что стало с текстом. Верни padding обратно.
Задание 4: придумай свои “идеальные” значения и запиши их в тетрадь.
