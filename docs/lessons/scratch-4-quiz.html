Викторина Средний

Создай игру-викторину на любую тему: география, математика, кино, природа. Ведущий задаёт вопросы, игрок вводит ответы, набирает очки. В конце — итоговый результат и оценка!

Чему научишься

  • Работать с переменными (счёт, номер вопроса)
  • Использовать блок «спросить и ждать» + переменную «ответ»
  • Применять условия (если ответ правильный / неправильный)
  • Работать со списками (хранить вопросы и правильные ответы)
  • Создавать обратную связь (правильно/неправильно)
  • Показывать итоговый результат с оценкой

План реализации

  1. Выбери тему: Придумай 8-10 вопросов на одну тему (или микс). Запиши вопросы и правильные ответы в тетрадь.
  2. Создай списки: Сделай 2 списка: «Вопросы» и «Ответы». Заполни их соответствующими данными (вопрос #1 — ответ #1 и т.д.).
  3. Переменные: Создай переменные «Счёт» (начинается с 0), «Номер вопроса» (начинается с 1), «Всего вопросов».
  4. Цикл вопросов: Сделай цикл «повторить (Всего вопросов) раз». Внутри: задай вопрос из списка по номеру, получи ответ.
  5. Проверка ответа: «Если ответ = элемент (Номер вопроса) списка Ответы» — увеличь счёт, скажи «Правильно!». Иначе — скажи «Неправильно, верный ответ: ...».
  6. Персонаж-ведущий: Добавь спрайт-ведущего, который задаёт вопросы, реагирует на ответы (меняет костюм — радостный/грустный).
  7. Итоговый экран: После всех вопросов покажи результат: «Ты набрал X из Y!». Добавь оценку: 5 — все правильно, 4 — одна ошибка, и т.д.
  8. Оформление: Добавь звуки (правильно/неправильно), фоновую музыку, красивый фон.

Критерии оценки

  • Минимум 8 вопросов
  • Вопросы и ответы хранятся в списках
  • Есть подсчёт очков (переменная «Счёт»)
  • Есть обратная связь — правильно/неправильно после каждого вопроса
  • В конце показывается итоговый результат
  • Есть персонаж-ведущий с анимацией

Идеи для улучшения

  • Добавь таймер на каждый вопрос (10 секунд на ответ)
  • Сделай случайный порядок вопросов (перемешивание списка)
  • Добавь подсказки (50/50 — убрать неправильные варианты)
  • Сделай таблицу рекордов (лучший результат сохраняется в переменную)
