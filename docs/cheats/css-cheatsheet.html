1. Что такое CSS

HTML отвечает за то, что есть на странице: текст, заголовки, картинки, списки.

CSS отвечает за то, как всё выглядит: цвета, фон, размер текста, рамки, кнопки, анимации.

Обычно мы думаем так: «Сначала я придумал структуру в HTML, потом нарядил её с помощью CSS».

Подсказки

  • CSS — это стили, оформление.
  • Стили можно менять, не трогая сам HTML.
  • В CodePen есть отдельное окошко для CSS — туда пишутся стили.

2. Синтаксис CSS

Одна «правилка» в CSS выглядит так:

селектор {
    свойство: значение;
}

Пример: покрасить все абзацы в красный:

p {
    color: red;
}

Разбор по строчкам

  • селектор — к чему применяем стили (например, p или .card).
  • свойство — что меняем (color, background-color и т.д.).
  • значение — на что меняем (red, 20px и т.п.).
  • Точка с запятой ; в конце строки — обязательна.

3. Цвета и HEX

В CSS цвет можно задавать по имени (red, blue, green), но чаще используют HEX-коды.

HEX-код — это цвет вида #RRGGBB, где:

  • RR — сколько красного;
  • GG — сколько зелёного;
  • BB — сколько синего.

00 — нет этого цвета, FF — максимум. 

/* примеры полезных HEX-цветов */
body {
    background-color: #f5f0e8; /* бежевый фон */
}

h1 {
    color: #2563eb;            /* синий заголовок */
}

p {
    color: #333333;            /* тёмно-серый текст */
}

Примеры HEX

#ffffff — белый
#000000 — чёрный
#ff0000 — красный
#00aaff — ярко-голубой
#f5f0e8 — бежевый фон

Для начала достаточно знать: цвет = # + 6 символов.

4. Текст и фон

Чаще всего мы настраиваем:

  • color — цвет текста;
  • background-color — цвет фона;
  • font-size — размер текста;
  • text-align — выравнивание текста.
body {
    background-color: #f5f0e8;
    font-family: system-ui, sans-serif;
}

h1 {
    color: #2563eb;
    text-align: center;
}

p {
    color: #333333;
    font-size: 16px;
}

Что здесь происходит?

  • body — фон и шрифт для всей страницы.
  • h1 — заголовок по центру, синий.
  • p — обычный текст, удобный для чтения.

Главное правило: не делай яркий текст на ярком фоне — должно быть контрастно и спокойно для глаз.

5. Блоки, отступы и рамки

Для красивых «карточек» полезны:

  • padding — отступ внутри блока;
  • margin — отступ снаружи;
  • border — рамка;
  • border-radius — скруглённые уголки;
  • box-shadow — тень.
.card {
    background-color: #ffffff;
    padding: 16px;
    margin: 16px 0;
    border-radius: 12px;
    border: 1px solid #e5e7eb;
    box-shadow: 0 6px 14px rgba(15,23,42,0.08);
}
Это пример «карточки». У неё есть фон, скруглённые уголки, рамка и лёгкая тень.

Подсказки

  • padding: 16px; — текст не прилипает к краям.
  • margin: 16px 0; — расстояние между карточками.
  • border-radius: 12px; — «пилюльки», а не квадраты.
  • box-shadow — блок будто приподнят над фоном.

6. Анимации и hover

В CSS можно делать простые анимации без JavaScript:

  • transition — плавный переход при наведении;
  • :hover — состояние «мышка наведена»;
  • @keyframes — собственная анимация (например, мигающая свечка).
И на меня тоже

Попробуй навести мышку на кнопку и на блок. Свечка мигает сама — это анимация через @keyframes.

Как это выглядит в CSS

.demo-button {
    transition: transform 0.2s, box-shadow 0.2s;
}

.demo-button:hover {
    transform: translateY(-3px) scale(1.03);
    box-shadow: 0 8px 18px rgba(37,99,235,0.4);
}
@keyframes flame {
    0%   { transform: translateY(0) scale(1); }
    50%  { transform: translateY(-2px) scale(1.05); }
    100% { transform: translateY(0) scale(0.95); }
}

Главное — сначала сделать обычный вид, а потом добавить к нему плавные эффекты.
