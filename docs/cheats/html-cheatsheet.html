Скелет HTML-страницы и ! + Tab

У любой страницы есть «скелет» — основные теги. В VS Code можно набрать ! и нажать Tab — и этот скелет появится сам.

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя первая страница</title>
</head>
<body>

    <!-- Здесь будет содержимое страницы -->

</body>
</html>
Делай отступы с помощью Tab, когда вкладываешь теги внутрь других — так код легче читать.
Текст

Текст: <h1>, <p>, <b>, <i>

<h1> — главный заголовок страницы.

<p> — абзац текста.

<b> — жирное выделение.

<i> — наклонный текст.

<h1>Мои любимые цвета</h1>

<p>Это обычный абзац текста.</p>

<p>Здесь есть <b>жирное слово</b> и <i>наклонное слово</i> в одном предложении.</p>

Ссылки

Ссылка: <a href="...">

Ссылка делается с помощью тега <a> и атрибута href.

<a href="https://google.com">Перейти в Google</a>

Картинки

Картинка: <img>

<img> вставляет картинку. У него нет закрывающего тега.

<img src="https://via.placeholder.com/150x100/87CEFA/ffffff?text=Blue"
     width="150"
     alt="Голубой цвет">
Списки

Список: <ul>, <ol> и <li>

<ul> — список с точками (маркированный).

<ol> — список с цифрами (нумерованный).

<li> — один пункт списка (используется и в <ul>, и в <ol>).

<ul>
    <li>Голубой</li>
    <li>Зелёный</li>
    <li>Фиолетовый</li>
</ul>

<ol>
    <li>Первый шаг</li>
    <li>Второй шаг</li>
    <li>Третий шаг</li>
</ol>

Пример маркированного списка (<ul>):

  • Голубой
  • Зелёный
  • Фиолетовый

Пример нумерованного списка (<ol>):

  1. Первый шаг
  2. Второй шаг
  3. Третий шаг
Таблица

Таблица: <table>, <tr>, <th>, <td>

Таблица состоит из строк <tr> и ячеек:

  • <th> — заголовок (жирный, сверху).
  • <td> — обычная ячейка.
<table>
    <tr>
        <th>Голубой</th>
        <th>Зелёный</th>
    </tr>
    <tr>
        <td>Цвет неба, мягкий и приятный.</td>
        <td>Цвет травы и леса.</td>
    </tr>
</table>

Форма

Форма и кнопка отправки: <form> и <input type="submit">

Форма — это блок, где есть поля для ввода и кнопка «Отправить».

<form action="#" method="post">
    <input type="text" name="color" placeholder="Ваш любимый цвет">
    <input type="submit" value="Отправить">
</form>
Запомнить:
  • action="#" — форма «отправляется», но мы остаёмся на этой же странице (для тренировки).
  • method="post" — стандартный способ отправки данных формы.
  • type="text" — поле для текста.
  • type="submit" — делает кнопку «Отправить».
  • value="Отправить" — текст на кнопке.
